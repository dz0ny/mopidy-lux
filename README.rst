`Mopidy <http://www.mopidy.com/>`_ extension with nice web UI.


Installation
============

Install by running::

    pip install Mopidy-Lux

Or, if available, install the Debian/Ubuntu package from `apt.mopidy.com
<http://apt.mopidy.com/>`_.


Usage
=============

#. Visit http://localhost:6680/lux
